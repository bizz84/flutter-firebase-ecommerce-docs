# Target of URI doesn't exist: 'package:flutter_gen/gen_l10n/app_localizations.dart'

Sometimes when installing new dependencies, all files that depend on `app_localizations.dart` throw an import not found error.

In such cases, the quickest solution is to **restart the analysis server**:

View > Command Palette > Dart: Restart Analysis Server
